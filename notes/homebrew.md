> The missing package manager for OS X

## Installation

    ruby -e "$(curl -fsSkL raw.github.com/mxcl/homebrew/go)"

### Upgrade Homebrew

    brew update

## Usage

### Get Help

    brew help

### Search for Package

    brew search ruby

### Install Package

    brew install wget

### Show Outdated Packages

    brew outdated

### Upgrade Package

    brew upgrade wget

### Uninstall Package

    brew uninstall wget
